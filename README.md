# AegisWeb
AegisWeb is a secure web application focused on practical cybersecurity implementation with a basic AI-assisted mechanism to detect suspicious activities.

## Project Domain
- Cybersecurity
- Web Development
- AI / Machine Learning

## Objectives
- Implement secure authentication and authorization
- Apply secure coding practices
- Detect suspicious or abnormal user activities
- Understand real-world cybersecurity concepts
